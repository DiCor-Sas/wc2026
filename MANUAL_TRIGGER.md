# Manual Pipeline Trigger

Use this if the dashboard has not updated
in the last 3 hours during the tournament.

## From your phone in 30 seconds:
1. Go to github.com/DiCor-Sas/wc2026
2. Tap the Actions tab
3. Tap "Auto Update WC 2026 Dashboard"
4. Tap "Run workflow" dropdown button
5. Tap the green "Run workflow" button

The pipeline takes about 15 minutes.
The footer timestamp will update when done.

## Pre-match trigger (75 min before kickoff):
Same steps above. Run manually if the
lineup badge on the match card still shows
"STARTING XI PENDING" at kickoff minus
60 minutes.
